# Font Awesome icons

Some Font Awesome (opens new window) webfont icons are also included to the bundle.

You can use those curated FA icons by just adding .i-{icon-name} class, eg.:

<i class="i-shopping-bag"></i>

# Change icons fonts

You can use Line Awesome (opens new window) in place of Font Awesome with easy by setting the Sass variable:

$icon-font: "line-awesome";
